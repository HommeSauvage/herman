## UI Modes

Herman has two distinct UI modes. Every UI feature must target **Rookie Mode**, **Normal Mode**, or **both**. See [MODES.md](MODES.md) for the decision framework. If you are unsure where a feature belongs, ask before implementing.

Desktop visual language (widths, tokens, shared primitives): [apps/desktop/DESIGN.md](apps/desktop/DESIGN.md).

## Deskktop app

The desktop app is made with electrobun. If you are modifying files for the electrobun part (desktop UI, logic and others) see [apps/desktop/AGENTS.md](apps/desktop/AGENTS.md)
